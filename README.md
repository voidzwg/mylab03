### 什么是 StateMachine 模型？
状态机（StateMachine）模型是一种行为建模方法，用来描述对象或系统在不同状态下的行为转换。

基本组成：

- StateMachine：状态机本体

- State：状态，表示系统在某个时刻的情形

- Transition：状态之间的迁移，定义触发条件和目标状态

- Event：触发迁移的事件

- （可选）Action、Guard 等元素
